# 文本识别示例（ML Kit）

这是一个使用 Google ML Kit 的端侧文本识别示例。应用支持从相册选择图片，或使用内置的 5 个样例（3 张打印体、2 张手写风格）。

## 包含内容
- ML Kit 端侧文本识别
- 相册选择器（无需运行时权限）
- 5 个运行时生成的内置样例图片

## 运行方式
1. 使用 Android Studio 或 JetBrains IDE 打开工程。
2. 同步 Gradle。
3. 在设备或模拟器上运行 `app` 配置。

## 内置样例
应用运行时会生成 5 张图片：
1. 打印体发票片段
2. 打印体会议便签
3. 打印体物流标签
4. 手写风格待办清单
5. 手写风格菜谱便签

## 说明
- 若要测试真实手写，建议使用相册选择器加载自己的照片。
- 识别结果会以纯文本形式显示。
