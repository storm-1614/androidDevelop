## Android 四大组件
### 活动 Activity
- Activity是最基本的Android应用程序组件，应用程序中，一个Activity通常就是一个单独的屏幕。每个Activity都被实为一个独立的类，并且从Activity基类继承而来，Activily类会提供视图控制组件的用户接口，并对事件作出响应，大多数应用程序都是由多个Activity组成的。  

### 广播消息接收器 (BroadcastReceiver)
